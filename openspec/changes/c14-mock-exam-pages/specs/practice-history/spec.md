## ADDED Requirements

### Requirement: Differentiate Mock Exams in History
The system SHALL display a "Mock" tag and the pass/fail status for mock exam sessions in the practice history list.

#### Scenario: Mock exam in history list
- **WHEN** user views the Practice History page
- **THEN** mock exam sessions are clearly labeled with a "Mock Exam" tag and show if the user passed or failed (based on passing score)
