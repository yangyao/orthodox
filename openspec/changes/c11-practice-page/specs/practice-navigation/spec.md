## ADDED Requirements

### Requirement: Previous/Next Navigation
The practice page SHALL allow users to navigate between questions using "Previous" and "Next" buttons.

#### Scenario: Navigating to next question
- **WHEN** user taps the "Next" button
- **THEN** the page transitions to the next question in the session

### Requirement: Answer Sheet Toggle
The practice page SHALL provide an "Answer Sheet" button that toggles a grid view of all questions.

#### Scenario: Jumping to a specific question
- **WHEN** user opens the answer sheet and taps on question number 15
- **THEN** the practice page jumps to and displays question 15
