## ADDED Requirements

### Requirement: Manage Personal Notes
The system SHALL allow users to create, read, update, and delete a personal note for any question.

#### Scenario: Adding a note
- **WHEN** user enters text into the notes field for a question and saves
- **THEN** the system stores the note for that question and user

### Requirement: View Questions with Notes
The system SHALL allow users to browse a list of all questions for which they have written a note.

#### Scenario: Browsing questions with notes
- **WHEN** user navigates to the "My Notes" page
- **THEN** the system displays a list of questions that have personal notes
