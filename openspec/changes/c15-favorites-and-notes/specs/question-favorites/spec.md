## ADDED Requirements

### Requirement: Toggle Favorite Status
The system SHALL allow users to toggle (add or remove) a "favorite" status for any question.

#### Scenario: Favoriting a question
- **WHEN** user taps the favorite icon for a question
- **THEN** the system updates the favorite status for that question and user

### Requirement: View Favorites List
The system SHALL allow users to browse a list of all favorited questions.

#### Scenario: Browsing favorites
- **WHEN** user navigates to the favorites page
- **THEN** the system displays a list of favorited questions, grouped or filtered by question bank
