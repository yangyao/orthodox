## ADDED Requirements

### Requirement: Home Page Banners
The home page SHALL display a carousel of promotional banners at the top.

#### Scenario: Displaying banners
- **WHEN** the home page is loaded
- **THEN** the system fetches and displays a list of active banners in a swiper component

### Requirement: Quick Entry Navigation
The home page SHALL provide a grid of icons for quick access to key features like "Question Bank Practice", "Mock Exams", etc.

#### Scenario: Navigating to features
- **WHEN** a user taps a quick entry icon
- **THEN** the system navigates the user to the corresponding page or feature

### Requirement: Featured Question Banks
The home page SHALL display a list of recommended or featured question banks.

#### Scenario: Displaying featured banks
- **WHEN** the home page is scrolled down
- **THEN** the system displays a curated list of question banks with their titles and cover images
