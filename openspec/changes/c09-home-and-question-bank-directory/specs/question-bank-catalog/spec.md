## ADDED Requirements

### Requirement: Search Question Banks
The catalog SHALL provide a search bar at the top for users to search for question banks by name.

#### Scenario: Searching by name
- **WHEN** user enters a keyword in the search bar
- **THEN** the system filters and displays matching question banks

### Requirement: Category Navigation
The catalog SHALL allow users to browse question banks by category using a sidebar or tab-based navigation.

#### Scenario: Switching categories
- **WHEN** user selects a different category
- **THEN** the system updates the list to show question banks belonging to that category

### Requirement: Question Bank List
The catalog SHALL display a list of question banks with relevant metadata like icon, title, number of questions, and update date.

#### Scenario: Displaying list
- **WHEN** a category is selected or a search is performed
- **THEN** the system renders a list of cards representing each question bank
